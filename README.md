Firefox extension that detects whether an image is ai generated or real.

It works best when detecting stable diffusion, DALL-E or midjourney images.

![2026-01-0316-40-55](https://github.com/user-attachments/assets/3b9c5b54-c857-4eeb-a5fc-d7d455bba3c4)

The model used is a vision transformer with mixture of experts in every other block.
